static< T > T rand ( final Set< T >  set)
static

Given a set of any type, returns an arbitrary element, using an equal distribution. Generics are used so that the returned element will have the same type as the set's elements have.

NOTE: This is not very efficient. If you need to do this on large sets several times per second, consider copying the set contents to an array, then call rand() on this array.

Parameters
