Detailed Description

This enum describe one of the possible state of the client.

When the client starts it has no state, once it connects to server it is assigned CONNECTION_ACCEPTED state until it logins successfully so it is moved to LOGIN_COMPLETE.

As soon as it chooses a character it is moved to GAME_BEGIN state which it will only leave when logout is accepted, LOGOUT_ACCEPTED.
