Detailed Description

QUEST: Daily Monster Kill Quest.

PARTICIPANTS: Mayor some creatures

STEPS: talk to Mayor to get a quest to kill one of a named creature class kill one creature of that class tell Mayor that you are done if after 7 days you were not able to kill the creature, you have an option to get another quest

REWARD: - xp - 5 karma

REPETITIONS: - once a day
